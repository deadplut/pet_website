# pet_website

## Подготовка окружения и установка зависимостей.

```bash
python3.12 -m venv .venv
source .venv/bin/activate
```

### Следующие команды выполняются для установки зависимостей
```(.venv) $ 
pip install -U pip
pip install -r requirements.txt
```

### Запуск сайта
```(.venv) $ 
pip install -U pip
pip install -r requirements.txt
```
